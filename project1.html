To complete this project you will need:

** This tutorial assumes you are starting on Windows

  1. USB 3.0 Stick Type-A (preferably 64GB & up)
  2. A funtional computer
  3. Stable internet connection
  4. ~8 minutes for configuration

Set persistence amount (can be updated later)

- I chose to split my drive storage 50/50. In my 256GB drive, the persistence is set to 128GB, which means I have 128GB remaining for regular thumb drive storage too. Persistence is required to save and store data like on a regular computer. Otherwise, everything you've accomplished on the drive will be wiped each time you eject it.

Conclusion

For this project, I wrote down the steps I took so that it would be easier to replicate the process in case I ever forgot, not intending to release the recipe. Now you can safely open Ubuntu from any device that takes USB 3.0 type-A drives! I recommend to set the passwords, regular and admin, as required if you plan on plugging this into a device that is not exclusivly yours. Once inside, your computer data should be secured by BitLocker and will need the key to see everything that it normally does.
